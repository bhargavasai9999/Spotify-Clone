# Spotify Clone

## Description

A simple Spotify clone with basic features.

## Table of Contents

- [Installation]
- [Usage]
- [Features]
- [Contributing]
- [License]

## Installation

git clone https://github.com/bhargavasai9999/Spotify-Clone.git
- cd Spotify-Clone
- npm install
- npm start

## Features

- User authentication
- Daily Feed and Categories\
- Trending
- Library 
- Playing and pausing songs
- Signout
